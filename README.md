# 🌏 千木社

## 欢迎访问 千木社

千木社原名 “Fuqing博客” ，成立于2022年，七月末更名为千木社。

千木社是由社长本人直接运营的公益向服务站。致力于将已有资源的利用率最大化，将值得共享的资源无偿共享。

现阶段对外提供免费网络服务。

> 千木社现已归档且削减大量服务，但这并不代表千木社运营即将到头，社长将尽量继续维护下去。
