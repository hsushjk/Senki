# 千木社

## 欢迎访问 千木社

千木社原名 “Fuqing博客” ，成立于2022年，七月末利用现有的个人名下资源转型为公益站，更名为千木社。

千木社是由 Fuqing本人直接运营，不受Milk Cap 365管理的公益站。致力于将已有资源的利用率最大化，将成品资源无偿共享，将信息差所带来的内耗减小。打造安全至上，稳定兼顾，性能拉满的使用体验。

现阶段对外提供网信，树洞等免费服务。
