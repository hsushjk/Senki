# 🌐 Universal Network Interface(通用网络接口)

### 以下为现存的网络接口：

#### hk-net1\[香港网01]:

域：hk-net1.senki.top

IPv6：无

最大带宽：35Mbps

标准带宽：30Mbps

Ping：55ms

抖动：0.5ms

丢包率：< 1%

备注：主接口

#### hk-net2\[香港网2]：

域：hk-net2.senki.top

IPv6：支持

最大带宽：5Gbps

标准带宽：255Mbps

Ping：130ms

抖动：1.20ms

丢包率：0%\~35%

备注：大带宽
