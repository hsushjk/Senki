# Senki-NTP

网络时间协议（英语：Network Time Protocol，缩写：NTP）是在数据网络潜伏时间可变的计算机系统之间通过分组交换进行时钟同步的一个网络协议。自1985年以来，NTP是目前仍在使用的最古老的互联网协议之一。

Senki-KMS为千木社自建的NTP服务器，用于时间同步，矫正客户机的时钟。支持1ms的时间精度。
